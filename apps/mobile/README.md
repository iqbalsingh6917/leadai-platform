# 📱 Mobile App — LeadAI Platform

> **Status:** Planned for Phase 4 (Month 15+)

The LeadAI Platform mobile app will bring the most critical sales and marketing management features to iOS and Android devices.

## Technology

**Planned stack:** React Native (with Expo) or Flutter — final decision in Phase 3.

| Consideration | React Native | Flutter |
|--------------|-------------|---------|
| Code sharing with web | ✅ (React) | ❌ |
| Performance | Good | Excellent |
| UI consistency | Good | Excellent |
| Community | Large | Growing |

## Planned Features

| Feature | Priority |
|---------|----------|
| Push notifications for hot leads (AI-detected) | P0 |
| Unified Lead Inbox (view + respond) | P0 |
| Pipeline view (visual stages) | P0 |
| Quick actions — Call, WhatsApp, Email from one tap | P0 |
| AI Chat interface (ask anything about your pipeline) | P1 |
| Analytics summary (daily/weekly digest) | P1 |
| Deal updates and activity notifications | P1 |
| WhatsApp message composer with AI suggestions | P1 |
| Campaign performance overview | P2 |
| Barcode/QR scan for business card lead capture | P2 |
| Voice notes for lead activities | P2 |

## Design Principles

- **Mobile-first**: Not a port of the web app; designed specifically for mobile workflows
- **Offline-first**: Key data cached locally, sync when connected
- **Minimal taps**: Critical actions (call, WhatsApp) reachable in 1–2 taps
- **Smart notifications**: AI determines notification priority, no notification spam
